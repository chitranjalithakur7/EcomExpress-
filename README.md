
#  Ecom Express Power BI Dashboard

This Power BI dashboard analyzes e-commerce sales data consisting of orders, customers, products, pricing, ratings, and location details (states). The dataset captures time-based transactions, so sales and revenue trends change over different periods.

The dashboard provides dynamic insights such as total revenue, number of orders, average order value (AOV), top-performing products, category-wise sales distribution, and state-wise performance. All calculations are built using DAX measures, ensuring that insights automatically update when new data is added or refreshed. Time intelligence features (Year and Quarter analysis) help in understanding sales trends over time without relying on fixed values.

## Tools Used

Power BI, DAX, Data Modeling, Data Visualization
